Our product advantages

  • Camera

    Camera will be turn off if no person is detected.

  • Power

    Because camera is not always on, we can achieve lower power consumption.

  • Network

    We only transfer image one by one, so network traffic can be considered minimal.

  • Server

    Image will be organize by server script and categorize into different folder.

  • Space

    We can even shrink images to minimize storage requirement.

  • Price

    We only need two sensors,which are ir and camera, and a raspberry pi.
